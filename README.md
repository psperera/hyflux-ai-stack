# hyflux-ai-stack
HyFlux AI Stack: A local, Docker-based AI research and RAG platform for Apple Silicon. Includes Open WebUI, hybrid search (SERPAPI/Bing/Tavily/SearXNG), GPT-Researcher, CryoSim-X, Stable Diffusion, notebooks, and a full multi-DB RAG engine with Meilisearch, Qdrant, and pgvector. Ideal for deep-tech R&amp;D.
