# HyFlux AI Stack — Installation Guide

This guide covers the complete setup of the HyFlux AI Stack on **macOS Apple Silicon** using Docker.

---

## 1. Requirements

1. **Docker Desktop**  
   https://www.docker.com/products/docker-desktop/

2. Docker → Settings → Resources:
   - Memory ≤ **48 GB**
   - 8+ CPUs recommended

3. Install **Git** and **Python 3** on the host.

---

## 2. Clone

```bash
git clone https://github.com/psperera/hyflux-ai-stack.git
cd hyflux-ai-stack
